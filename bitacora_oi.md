### proceso

0. definir un dataset (en base a los distintos sets que hay en el repositorio Selva Fría)
1. creación de un programa isadora que permita disponer de un dataset
2. reproducir algun elemento del dataset
3. seleccionar y reproducir en tiempo real cualquier elemento del dataset
4. implementar una cola de reproduccion de un solo elemento simultáneamente, que permita tener en un estado de preselección cualquier elemento del dataset
5. reproducir el elemento preseleccionado en la cola de reproducción
6. vaciar la cola de reproducción
7. creación interfaz gráfica (UX) del programa
